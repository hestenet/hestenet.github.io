INTRODUCTION
------------

Html Head module allows to add all html tags inside head tag.

INSTALLATION
------------

* Install as you would normally install a contributed Drupal module. Visit:
https://www.drupal.org/docs/8/extend/installing-modules
for further information.

CONFIGURATION
-------------
* Add Html Tags inside head tag, settings on the Administer -> Configuration
-> Development -> HTML HEAD.
* Config translation(admin/config/development/html_head/translate).
